# CRUD-PAM-II---Tuma-A-Per-odo-Manh-

Para usar o projeto baixe o VSCode <br>
Abra a pasta extraida para no VSCode e use os seguintes comandos no terminal <br>
CD backend <br>
npm i -g json-server <br>
npm i -g localtunnel <br>
Para iniciar o banco de dados virtual use: <br>
npx json-server --watch db.json --port 3000 <br>
Agora para inicializar o local tunnel duplique o terminal e use: <br>
npx lt --port 3000 <br>
Coloque o link entregue pelo Local Tunnel no ConfigApi.json substituindo o Link antigo <br>
Abra outro terminal e vá para a pasta meu crud com o comando: <br>
cd frontend/meucrud <br>
Agora baixe os dados para iniciar o projeto com o comando: <br>
npm i <br>
Para iniciar o projeto agora use: <br>
npx expo start
